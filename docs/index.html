Uygulama Yetenekleri

Almanca kelime dağarcığınızı hızlıca genişletmek ve öğreniminizi daha verimli kılmak için geliştirilen araçlar.

Aralıklı Tekrarlama (SM-2)

SuperMemo SM-2 bilimsel algoritmasını kullanan FlaschDeck, kelimeleri unutmamanız için en uygun zaman dilimlerinde tekrarlamanızı hatırlatır.

Gemini Yapay Zeka Desteği

Yapay zekanın gücüyle tek tıkla kelimelerinize Almanca örnek cümleler, Türkçe çeviriler ve detaylı dil bilgisi (gramer) açıklamaları ekleyin.

Görsel Artikel Hafızası

İsim artikellerini (der, die, das) renk kodlarıyla ezberleyin. der = Mavi, die = Kırmızı, das = Yeşil olarak işaretlenerek zihninizde yer edinir.

Quiz Arenası

Öğrendiğiniz kelimeleri çoktan seçmeli testlerle veya eğlenceli kart eşleştirme oyunları ile test ederek bilgilerinizi pekiştirin.

Gelişmiş İstatistikler

Günlük kelime tekrar sayınızı, quiz başarı oranlarınızı ve öğrenim grafiklerinizi takip ederek ilerlemenizi gözlemleyin.

Deste İçe/Dışa Aktarma

Hazırladığınız Almanca kelime destelerini JSON formatında kolayca dışarı aktarabilir, arkadaşlarınızın destelerini de hızlıca içe aktarabilirsiniz.

Kurulum Kılavuzu

İndirdiğiniz uygulamayı bilgisayarınızda nasıl çalıştıracağınızı öğrenin.

Windows

  • FlaschDeck.Setup.x.y.z.exe dosyasını indirin.
  • İndirilen dosyaya çift tıklayarak kurulum sihirbazını başlatın.
  • Kurulum tamamlandığında masaüstünüzde FlaschDeck kısayolu oluşacaktır. Uygulamayı çalıştırın ve öğrenmeye başlayın.

macOS

  • FlaschDeck-x.y.z.dmg dosyasını Mac cihazınıza indirin.
  • Dosyaya çift tıklayarak DMG imajını açın.
  • FlaschDeck ikonunu tutup Applications (Uygulamalar) klasörüne sürükleyin.
  • Uygulamalar menüsünden FlaschDeck'i başlatın. (İlk çalıştırmada güvenlik uyarısı alırsanız sistem ayarlarından izin verebilirsiniz.)

Linux

  • FlaschDeck-x.y.z.AppImage dosyasını bilgisayarınıza indirin.
  • İndirilen dosyaya sağ tıklayıp Özellikler > Erişim İzinleri kısmından "Çalıştırılabilir olarak izin ver" seçeneğini aktif edin.
  • Terminalden çalıştırmak isterseniz chmod +x FlaschDeck-x.y.z.AppImage komutunu çalıştırın.
  • Çalıştırılabilir izinlerini verdikten sonra dosyaya çift tıklayarak uygulamayı anında açabilirsiniz.
